最大公約数

用途
ユークリッドの互除法のアルゴリズムが最適
ARCのAでよく使う印象

mathのgcdでもいい
そっちのほうが配列も使える

計算量
O(log (A+B))

テンプレ
------------------------------------------------------------------
def gcd(a,b):
    while a > 0 and b > 0:
        if a > b:
            a = a % b
        else:
            b = b % a

    return max(a, b)
